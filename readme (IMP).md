🚀 Getting Started
Follow the steps below to run Zingo on your local machine 👇

1️⃣ Download my Whole Repo and open it in your preferred IDE

2️⃣ Before running the app, make sure all the required npm packages are installed.
To do this, open the terminal inside your project folder and run: 
                        npm install

3️⃣ Zingo uses a proxy server to handle CORS (Cross-Origin Resource Sharing) issues while fetching APIs.
Before running the project, make sure you take access to the proxy server using the link below 👇

                🔗 https://cors-anywhere.herokuapp.com/corsdemo 

⚠️ Note: This step is important — without proxy access, some API calls may fail due to browser CORS restrictions.

4️⃣ Run the Project:
After installing dependencies and enabling proxy access, start the development server using: npm run dev
